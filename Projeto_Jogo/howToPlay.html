PREPARE-SE PARA SER ATACADO

Zombie é um jogo em que o player terá de matar o máximo de zumbis possíveis.
Quanto mais você os destrói, mais rápidos eles aparecem.
Se os zumbis alcançarem o player... GAME OVER. SOBREVIVA SE PUDER!!!

Os comandos do jogo são os seguintes:
Tecla Enter -> Dá início e pausa o jogo.
Setas para a esquerda -> Movimento do player para esquerda.
Setas para a direita -> Movimento do player para direita.
Barra de espaço -> Atira contra os zumbis.
Obs: Ao perder o jogo ('Game Over'), a tela inicial aparecerá novamente.
